## 🔐 About HackersBun

**HackersBun** is a professional-grade cybersecurity and AI news aggregator designed for security analysts, Threat Intelligence teams, and SOC professionals. Stay informed about the latest breaches, vulnerabilities, attacks, and cyber threats - all in one powerful dashboard.

### Built for Cyber Security Professionals

- ✅ Real-time threat intelligence
- ✅ CVE Detection & Highlighting
- ✅ Search & Filter capabilities
- ✅ Export for reports
- ✅ Bookmark critical articles
- ✅ Time-based filtering

## ✨ Features for Security Analysts

### 🔍 Smart Search
Search across all news for:
- CVE numbers (CVE-2024-XXXX)
- Keywords: breach, malware, ransomware, APT, phishing
- Company names: Microsoft, Cisco, CrowdStrike

### 📅 Time Filters
Quickly filter news by:
- **Today** - Last 24 hours
- **Week** - Last 7 days  
- **Month** - Last 30 days
- **All** - All available

### 🏷️ CVE Detection
- Auto-detects CVEs in titles
- Critical severity badges (red)
- High severity badges (orange)

### 📥 Export to CSV
One-click export for:
- Daily reports
-IOC extraction
- Threat intel sharing

### ⭐ Bookmark System
- Save critical articles
- Persists in browser storage
- Quick access modal
- Export saved bookmarks

### ⚡ Other Features
- 🔄 Auto-refresh (30 seconds)
- 🌙 Dark/Light theme toggle
- 📱 Responsive design
- 🖱️ Open/Copy on hover

## 🛠️ Technical Details

- **Single File**: self-contained HTML (no dependencies)
- **No Backend**: All data from RSS feeds via proxy
- **Browser Storage**: Bookmarks saved locally
- **Proxies**: Multi-proxy fallback for reliability
